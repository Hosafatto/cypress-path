# How to Install [Top](#cypress-path-course)
1. Create `package.json`. (Run `npm init -y`)
2. Install Cypress. (Run `npm install cypress --save-dev`)
3. That's it!

# How to Run Cypress [Top](#cypress-path-course)
1. Run `npx cypress open`
