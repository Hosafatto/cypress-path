## How to Install
1. Create `package.json`. (Run `npm init -y`)
2. Install Cypress. (Run `npm install cypress --save-dev`)
3. That's it!

## Write your first test
1. Create a folder and name it `e2e`.
2. Create a file ´*.cy.js´ in e2e folder. (Example: firstTest.cy.js)
3. Write your tests!!!
